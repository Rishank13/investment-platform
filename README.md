# 📊 Investment Analytics Platform

## 📌 Overview
A backend application built using Spring Boot that simulates a real-world investment analytics system focused on mutual funds.

The platform ingests NAV (Net Asset Value) data, stores historical price data, computes financial metrics, and ranks assets based on performance.

The project follows a layered architecture and demonstrates backend system design, data ingestion, and analytics processing.

🚧 This project is currently under development and continuously being improved.

---

## 🚀 Modules
- Asset Module
- Pricing (Price History) Module
- Analytics Module
- Ranking Module
- Scheduler (Data Ingestion)
- Common (Exception Handling, Utilities)
- Config Module

---

## 🛠️ Tech Stack
- Java 17
- Spring Boot
- Spring MVC
- Spring Data JPA
- Maven
- MySQL / PostgreSQL

---

## ⚙️ Features
- Asset management
- NAV data storage
- Financial analytics (CAGR, Volatility)
- Asset ranking and filtering
- Scheduled data ingestion
- Modular and scalable architecture
